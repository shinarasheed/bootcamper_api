#Bootcamper API

> Backend API for the Bootcamper api which is a bootcamp directory website

## Usage

Rename "config/config.env.env" to "config/config.env" and update the values/settings with your own

## Install Dependencies

```
npm install
```

## Run App

```
# Rub in dev mode
npm run dev


# Run in prod mode
npm start
```
